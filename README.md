# 动态简历(animate_resume)

## 在线观看 view online

https://qishaoxuan.github.io/animate_resume/

## 关于 about

- 文字加载基于 ES6 + 异步promise封装
- 字体样式参考 prismjs
- prismjs + markedjs 处理代码转换
- 移动端使用了 better-scroll 优化体验
- 打包使用了 gulp + rollup

## 本地查看
```bash
  git clone git@github.com:QiShaoXuan/animate_resume.git
  cd animate_resume
  npm install
  npm run dev
```

## 修改
style 和 resume 在 `src/load/` 下的对应js中

## 打包
```bash
  npm run build
```
